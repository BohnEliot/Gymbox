<p align="justify">
  
A GymBox egy olyan webalkalmazás, amely fantasztikus megoldást kínál a fitnesz és az edzés területén. Az alkalmazás célja, hogy egyszerűbbé és rugalmasabbá tegye az edzőeszközök használatát azok számára, akik szeretnének rendszeresen mozogni anélkül, hogy hagyományos edzőteremi tagságot kellene vállalniuk, vagy fix helyhez lennének kötve. A GymBox segítségével különböző méretű és felszereltségű konténereket lehet bérelni, melyeket magas minőségű, korszerű edzőgépekkel töltöttünk fel, így akár otthon, akár a munkahely közelében, vagy bármilyen más, számukra kényelmes helyszínen végezhetik el a testmozgást.
Felhasználóink számára lehetővé tesszük, hogy online gyorsan és egyszerűen válasszanak a különböző konténerek közül, amelyek eltérő edzésigényeket és felszereltségi szinteket kínálnak. A bérlés rugalmassága révén akár rövid időre is igénybe vehető egy-egy konténer, így a felhasználók saját időbeosztásuk szerint, stresszmentesen szervezhetik meg az edzéseiket.

A webalkalmazás indítása nem igényel különösebb informatika tudást, csupán az alábbi alkalmazások szükségesek hozzá: Xampp8.1.12 vagy újabb verzió, NodeJs v22.13.0 vagy újabb verzió, Composer 2.8.4 vagy újabb verzió, Visual Studio Code v1.96.0 vagy újabb verzió és egy általuk választott böngésző, amely naprakész és megbízható. Innentől már csak a helyes sorrendben kell indítanunk az alkalmazás egyes részeit.

Első lépés a backend beüzemelése. Ehhez szükséges telepíteni a Xampp-ot és a Composert. Megnyitjuk a xampp-control panelt és elindítjuk a Mysql és Apache modulokat, ez létrehozza a kapcsolatot a lokális adatbázissal, ahol az adatokat tároljuk. Második lépésként majd létrehozunk egy mappát pl.: Gymbox néven. A Gymbox mappában, ahol telepítve van az Angular CLI és a Laravel a Composerrel együtt, el kell helyezni a teljes backend és frontend mappákat, amelyek tartalmazzák az alkalmazás adatbázisát és felhasználói felületét. Harmadik lépésben elindítjuk a Visual Studio Code-ot és megnyitjuk benne a backend mappát, amely a php alapú Laravel keretrendszerben készült. Itt megnyitjuk a terminált (Ctrl + ö) és a következő parancsot írjuk be: php artisan migrate –seed. Ezzel feltöltjük a xampp-ba az adatbázist. Aztán elindítjuk a szervert, szintén a terminálban a php artisan serve paranncsal. Ezt követően a Visual Studio Code-ban megnyitjuk a frontend mappát (GymboxAngular), amely Angular keretrendszerben készült. Itt a terminálban átváltunk parancssorra, a powershell helyett. Ha még nem telepítettük az Angulart, akkor beírjuk a terminálba, hogy: npm install -g @angular/cli. Ezzel feltelepítjük az adott mappába a keretrendszert. Majd beírjuk azt a parancsot, hogy ng serve -o. Ezzel megnyitottuk és elindítottuk a webalkalmazást.
